# Ứng dụng điểm danh bằng mã QR

## Mô tả
Ứng dụng hỗ trợ điểm danh sinh viên bằng mã QR, gồm các chức năng: tạo mã QR, quét mã QR, xác thực sinh viên, thống kê điểm danh.

## Công nghệ
- Frontend: HTML, CSS, JavaScript, Bootstrap, QRCode.js, html5-qrcode
- Backend: Node.js, Express.js
- Database: MongoDB

## Cách sử dụng
1. Cài đặt Node.js, MongoDB
2. Chạy backend: `npm install` và `npm start` trong thư mục backend
3. Mở frontend trên trình duyệt
4. Push code lên GitHub: `git add .`, `git commit -m "init"`, `git push`

## Thư mục
- frontend/: Giao diện người dùng
- backend/: API và xử lý dữ liệu
- database/: Cấu hình MongoDB

## Liên hệ
Nếu cần hỗ trợ, liên hệ nhóm phát triển.
