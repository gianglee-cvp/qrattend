# Nguyên tắc dự án & AI Hook (Claw)

## 1. Đồng bộ Tài liệu (Git Hook AI)
- Mọi thay đổi code hoặc hoàn thành task phải được cập nhật ở `progress.md` và `CODEMAP.md`.
- File này (`claw.md`) là file tham chiếu các quy tắc chung.

## 2. Quy tắc phát triển (QRAttend)
- **Frontend:** Giữ nguyên sự đơn giản với Vanilla JS hoặc nâng cấp nhẹ nhàng nếu không có chỉ định (chủ yếu mobile-optimized).
- **Backend:** Node.js/Express với code sạch sẽ, chia route nếu cần.
- **Bảo mật:** Ưu tiên các tính năng chống gian lận ở cả Client và Server.
